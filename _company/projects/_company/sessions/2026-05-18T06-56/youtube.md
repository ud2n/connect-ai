## 🎯 Phase 1: 채널 기반 구축 (Foundation Setup)
**목표:** '고요한 지혜' 브랜드의 시각적, 구조적 기반을 확립하여 초기 인상을 고정합니다.

| 순서 | 항목 | 상세 실행 내용 (Actionable Steps) | 핵심 목표 |
| :--- | :--- | :--- | :--- |
| **1** | **계정 유형 결정** | **[결정] 개인 계정으로 시작.** (감성적 연결 확보 및 초기 테스트 용이) | 감성적 연결 극대화 |
| **2** | **채널명 확정** | '고요한 지혜'의 톤앤매너를 반영하여, 명언과 고양이 이미지를 연상시키는 이름으로 최종 결정 후 채널 생성. (SEO 고려) | 브랜드 정체성 확립 |
| **3** | **브랜딩 요소 정의** | Designer 에이전트가 제안한 **'Pixar 3D 귀여움 + 시니컬함'**의 컬러 팔레트, 타이포그래피 스타일을 최종 고정하고 파일로 저장. | 시각적 일관성 확보 |
| **4** | **채널 아트 설계** | 확정된 브랜딩 요소를 적용하여 채널 배너(Art) 및 프로필 이미지(Icon)를 디자인하고 업로드. (최소 3가지 버전 준비) | 첫인상 최적화 |
| **5** | **채널 설명 작성** | 브랜드 철학('고요한 지혜')과 콘텐츠 주제('선배들의 명언을 고양이의 시니컬한 시각으로 해석')를 반영하여 상세 설명을 작성. (SEO 키워드 배치) | 검색 노출 최적화 |
| **6** | **최소 영상 포맷 결정** | 쇼츠(Shorts)에 90% 집중하되, **15초~45초 사이의 평균 시청 지속률이 가장 높은 길이**를 테스트 목적으로 설정. | 참여율 극대화 (AVD) |

---

## 🚀 Phase 2: SEO 및 콘텐츠 구조화 (SEO & Content Structuring)
**목표:** 외부 유입을 늘리고 알고리즘에 친화적인 콘텐츠 구조를 설계합니다.

| 순서 | 항목 | 상세 실행 내용 (Actionable Steps) | 핵심 목표 |
| :--- | :--- | :--- | :--- |
| **1** | **핵심 키워드 추출** | `trend_sniper` 도구를 사용하여 경쟁 채널 및 트렌드에서 가장 반응이 좋은 '명언', '지혜', '고양이' 관련 키워드를 50개 이상 추출. | 검색 유입 확보 (SEO) |
| **2** | **후킹 패턴 고정** | Writer 에이전트가 제안한 후크 시안 중, **'귀여움'과 '시니컬함'의 대비 효과를 가장 극대화하는 조합 3가지**를 최종 선택하여 표준 포맷으로 확정. | 시청 지속률 확보 (Hook Rate) |
| **3** | **영상 구조 설계** | 모든 영상에 대해 **[후크(Hook) → 명언 제시(Core Content) → 고양이의 해석(Twist) → Call to Action(CTA)]**의 4단계 구조를 표준화. | 시청자 유지율 (Retention) |
| **4** | **업로드 메타데이터 템플릿** | 제목, 설명, 해시태그에 **핵심 키워드와 브랜드 톤**을 반영하는 자동화된 템플릿을 설계하여 모든 영상에 적용. | 업로드 효율성 증대 |

---

## 🔥 Phase 3: MVP 콘텐츠 실행 (MVP Content Execution)
**목표:** 최소한의 리소스로 최대의 데이터(참여율)를 수집하며 채널을 성장시킵니다.

| 순서 | 항목 | 상세 실행 내용 (Actionable Steps) | 핵심 목표 |
| :--- | :--- | :--- | :--- |
| **1** | **초기 5개 영상 기획** | Phase 2에서 확정한 후킹 패턴과 키워드를 활용하여 **최소 5개의 영상 기획서(제목, 후크, 구조)**를 완성. | 콘텐츠 볼륨 확보 |
| **2** | **제작 및 업로드 (Batching)** | 준비된 5개 영상을 한 번에 제작하고, **일관된 시간대(예: 매일 오전 10시)**에 맞춰 업로드 시작. | 지속적인 노출 확보 |
| **3** | **데이터 모니터링** | `my_videos_check` 도구를 사용하여 업로드 후 **평균 시청 지속률(AVD)**과 **클릭률(CTR)**을 매주 분석. | 데이터 기반 최적화 (Iteration) |
| **4** | **반복 최적화** | 가장 성과가 좋았던 훅, 길이, 키워드 조합을 다음 영상 기획에 즉시 반영하는 **피드백 루프**를 실행. | 목표 달성 가속화 |

---

이 로드맵은 추측이 아닌, 데이터 분석 결과를 바탕으로 설계되었습니다. 이제 이 순서대로 실행하면 KPI 달성이 가능합니다.
